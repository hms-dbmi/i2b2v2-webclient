# [SUMMARY] - Query Summary Visualization Module
Written by Nick Benik in 2025

## Description

---

The summary visualization module is used to display the current status of the query that is running or has run.
It describes list the `"Query Name"`, `"Query Status"`, and `"Run Time"` (which may be counting up over time). 
The visualization module also displays a `"Cancel Query"` button for canceling the query and a `"View Report"` button which
launches the query report window used to print the results of the query.

The module is designed to ingest the specialized `"INTERNAL_SUMMARY"` pseudo-breakdown that is generated by the QueryRunner subsystem.

### Definition within `breakdowns.json`
```json
{
  "INTERNAL_SUMMARY": {
    "SUMMARY": true
  },
  ...
}
```
