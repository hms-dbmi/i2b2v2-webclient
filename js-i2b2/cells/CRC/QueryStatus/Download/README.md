# [DOWNLOAD] - Download Visualization Module
Written by Nick Benik in 2025

## Description

---
The Download "visualization" module is used to allow end-users to download a breakdown's data as a CSV file. 

It has no configuration options.

### Definition within `breakdowns.json`
```json
{
  "PATIENT_GENDER_COUNT_XML": {
    "DOWNLOAD": true
  },
  "PATIENT_SEX_COUNT_SHRINE_XML": {
    "DOWNLOAD": true
  },
  "PATIENT_VITALSTATUS_COUNT_XML": {
    "DOWNLOAD": true
  },
  "PATIENT_VITALSTATUS_COUNT_SHRINE_XML": {
    "DOWNLOAD": true
  }, ...
}
```
