Waste Water Previous Query Visualizer

Back to Plugin

Workflow

Overlay previous queries with waste water data between specific dates.

  • Select a start date and end date for the waste water data and click "start visualization"
  • Drag and drop a previous query onto the rendered waste water data
    • If the query does not contain temporal constraints or date metadata overlapping with the selected waste water data, no previous query data will be shown
    • Users can drop a maximum of 3 previous queries
